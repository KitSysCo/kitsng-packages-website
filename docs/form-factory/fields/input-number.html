Input Number Field

The input-number field type provides a numeric input with spinner controls and formatting options.

Basic Usage

{
  controlType: 'input-number',
  colSize: 'col-span-12',
  options: {
    formControlName: 'quantity',
    label: 'Quantity',
    placeholder: 'Enter quantity',
    mode: 'decimal',
    min: 1,
    max: 100,
    step: 1,
    validators: {
      required: true,
      min: 1,
      max: 100
    }
  }
}

Options

Common Options

Option Type Default Description
formControlName string Required Name of the form control
label string - Label displayed above the field
placeholder string - Placeholder text
mode 'decimal' \ 'currency' 'decimal' Input mode
min number - Minimum value
max number - Maximum value
step number 1 Step increment/decrement
showButtons boolean false Show +/- buttons
buttonLayout 'stacked' \ 'horizontal' \ 'vertical' 'stacked' Button layout
prefix string - Prefix text (e.g., '$')
suffix string - Suffix text (e.g., 'kg')
minFractionDigits number - Minimum decimal places
maxFractionDigits number - Maximum decimal places
useGrouping boolean true Use thousand separators
incrementButtonClass string - CSS class for increment button
decrementButtonClass string - CSS class for decrement button
incrementButtonIcon string - Icon class for increment button
decrementButtonIcon string - Icon class for decrement button
readonly boolean \ function false Whether field is readonly
disabled boolean \ function false Whether field is disabled

Validation Options

validators: {
  required?: boolean;
  min?: number;
  max?: number;
  // ... other validators
}

Examples

Basic Number Input

{
  controlType: 'input-number',
  colSize: 'col-span-12',
  options: {
    formControlName: 'age',
    label: 'Age',
    placeholder: 'Enter age',
    min: 18,
    max: 120,
    validators: {
      required: true,
      min: 18,
      max: 120
    }
  }
}

Number with Spinner Buttons

{
  controlType: 'input-number',
  colSize: 'col-span-12',
  options: {
    formControlName: 'quantity',
    label: 'Quantity',
    min: 0,
    max: 1000,
    step: 1,
    showButtons: true,
    buttonLayout: 'stacked',
    validators: {
      required: true
    }
  }
}

Currency Input

{
  controlType: 'input-number',
  colSize: 'col-span-12',
  options: {
    formControlName: 'price',
    label: 'Price',
    placeholder: 'Enter price',
    mode: 'currency',
    prefix: '$',
    min: 0,
    minFractionDigits: 2,
    maxFractionDigits: 2,
    useGrouping: true,
    validators: {
      required: true,
      min: 0
    }
  }
}

Weight Input with Suffix

{
  controlType: 'input-number',
  colSize: 'col-span-12',
  options: {
    formControlName: 'weight',
    label: 'Weight',
    placeholder: 'Enter weight',
    suffix: ' kg',
    min: 0,
    max: 500,
    step: 0.1,
    minFractionDigits: 1,
    maxFractionDigits: 2,
    showButtons: true,
    validators: {
      required: true
    }
  }
}
