Input Field

The input field type provides a standard text input with support for text and password types.

Basic Usage

{
  controlType: 'input',
  colSize: 'col-span-12',
  options: {
    formControlName: 'username',
    label: 'Username',
    placeholder: 'Enter your username',
    type: 'text',
    validators: {
      required: true,
      minLength: 3,
      maxLength: 20
    }
  }
}

Options

Common Options

Option Type Default Description
formControlName string Required Name of the form control
label string - Label displayed above the field
placeholder string - Placeholder text
type 'text' \ 'password' 'text' Input type
class string - Additional CSS classes
autocomplete string 'off' HTML autocomplete attribute
readonly boolean \ function false Whether field is readonly
disabled boolean \ function false Whether field is disabled

Validation Options

validators: {
  required?: boolean;
  minLength?: number;
  maxLength?: number;
  pattern?: string;
  email?: boolean;

}

Examples

Text Input

{
  controlType: 'input',
  colSize: 'col-span-12',
  options: {
    formControlName: 'email',
    label: 'Email Address',
    placeholder: 'Enter your email',
    type: 'text',
    validators: {
      required: true,
      email: true
    }
  }
}

Password Input

{
  controlType: 'input',
  colSize: 'col-span-12',
  options: {
    formControlName: 'password',
    label: 'Password',
    placeholder: 'Enter your password',
    type: 'password',
    validators: {
      required: true,
      minLength: 8,
      pattern: '^(?=.[a-z])(?=.[A-Z])(?=.*\\d).+$' // At least one lowercase, uppercase, and digit
    },
    errorMessage: 'Password must be at least 8 characters with uppercase, lowercase, and number'
  }
}

Input with Custom Validator

{
  controlType: 'input',
  colSize: 'col-span-12',
  options: {
    formControlName: 'username',
    label: 'Username',
    placeholder: 'Enter username',
    validators: {
      required: true,
      minLength: 3,
      maxLength: 20,
      custom: (control) =&gt; {
        // Custom validator: username must start with a letter
        const value = control.value;
        if (value && !/^[a-zA-Z]/.test(value)) {
          return { invalidUsername: 'Username must start with a letter' };
        }
        return null;
      }
    }
  }
}

Conditional Readonly

{
  controlType: 'input',
  colSize: 'col-span-12',
  options: {
    formControlName: 'accountId',
    label: 'Account ID',
    readonly: (form: FormGroup) =&gt; {
      // Make readonly if account type is 'premium'
      return form.get('accountType')?.value === 'premium';
    }
  }
}

Conditional Visibility

{
  controlType: 'input',
  colSize: 'col-span-12',
  options: {
    formControlName: 'companyName',
    label: 'Company Name',
    visibleWhen: {
      field: 'userType',
      value: 'business',
      operator: 'equals'
    }
  }
}

Input with Autocomplete

{
  controlType: 'input',
  colSize: 'col-span-12',
  options: {
    formControlName: 'name',
    label: 'Full Name',
    placeholder: 'Enter your full name',
    autocomplete: 'name',
    validators: {
      required: true
    }
  }
}

Input with Change Handler

{
  controlType: 'input',
  colSize: 'col-span-12',
  options: {
    formControlName: 'searchTerm',
    label: 'Search',
    placeholder: 'Type to search...',
    ngModelChange: (value, formGroup, selectedOption, field) =&gt; {
      console.log('Search term changed:', value);
      // Trigger search, update suggestions, etc.
    }
  }
}

Layout Options

Horizontal Layout

{
  controlType: 'input',
  colSize: 'col-span-6',
  options: {
    formControlName: 'firstName',
    label: 'First Name',
    controlLayout: 'horizontal',
    labelColSize: 'col-3',
    controlColSize: 'col-9'
  }
}

Vertical Layout (Default)

{
  controlType: 'input',
  colSize: 'col-span-12',
  options: {
    formControlName: 'lastName',
    label: 'Last Name',
    controlLayout: 'vertical' // or omit for default
  }
}

Accessibility

The input field automatically includes:

  • Unique field IDs
  • ARIA labels
  • ARIA required attributes
  • ARIA invalid attributes
  • Error message linking via aria-describedby
