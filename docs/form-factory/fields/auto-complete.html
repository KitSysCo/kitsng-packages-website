Auto Complete Field

The auto-complete field type provides an autocomplete input that suggests options as you type.

Basic Usage

{
  controlType: 'auto-complete',
  colSize: 'col-span-12',
  options: {
    formControlName: 'search',
    label: 'Search',
    placeholder: 'Type to search...',
    suggestionOptions: [{id: 1, text: 'Apple'}, {id: 2, text: 'Banana'}],
    validators: {
      required: true
    }
  }
}

Options

Common Options

Option Type Default Description
formControlName string Required Name of the form control
label string - Label displayed above the field
placeholder string - Placeholder text
suggestionOptions any[] [] Static suggestions array
optionLabel string 'text' Property name for option label
optionValue string 'id' Property name for option value
minCharToStartSearch number 1 Minimum characters before search
delay number 300 Delay in ms before search
showDropdown boolean false Show dropdown button
forceSelection boolean false Force selection from list
allowNewValue boolean false Allow entering custom values not in list
onNewValue function - Callback when new value is entered
onOptionSelected function - Callback when option is selected

API Options (for dynamic loading)

Option Type Default Description
apiBaseUrl string - Base URL for API
entity string - Entity name for lookup
module string - Module name for lookup
version string - API version
query object - Additional query parameters
memoryCache boolean false Cache results in memory

Styling Options

Option Type Default Description
style object {} Inline style for container
inputStyle object {} Inline style for input
panelStyle object {} Inline style for dropdown panel
inputStyleClass string 'w-full' CSS class for input
panelStyleClass string - CSS class for dropdown panel

Examples

Static Suggestions

{
  controlType: 'auto-complete',
  colSize: 'col-span-12',
  options: {
    formControlName: 'country',
    label: 'Country',
    placeholder: 'Type country name...',
    suggestionOptions: [
      { code: 'US', name: 'United States' },
      { code: 'CA', name: 'Canada' }
    ],
    optionLabel: 'name',
    optionValue: 'code',
    minCharToStartSearch: 1
  }
}

Dynamic Suggestions with API

{
  controlType: 'auto-complete',
  colSize: 'col-span-12',
  options: {
    formControlName: 'user',
    label: 'Search User',
    placeholder: 'Type user name...',
    apiBaseUrl: 'https://api.example.com',
    entity: 'users',
    module: 'auth',
    version: 'v1',
    optionLabel: 'fullName',
    optionValue: 'id',
    minCharToStartSearch: 3,
    delay: 500
  }
}

Allow New Values

{
  controlType: 'auto-complete',
  colSize: 'col-span-12',
  options: {
    formControlName: 'tags',
    label: 'Tags',
    placeholder: 'Add tags...',
    suggestionOptions: [{id: 'js', text: 'JavaScript'}],
    allowNewValue: true,
    onNewValue: (value, form, field) => {
      console.log('New tag added:', value);
    }
  }
}
