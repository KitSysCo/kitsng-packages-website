Textarea Field

The textarea field type provides a multi-line text input for longer content.

Basic Usage

{
  controlType: 'textarea',
  colSize: 'col-span-12',
  options: {
    formControlName: 'description',
    label: 'Description',
    placeholder: 'Enter description',
    rows: 5,
    validators: {
      required: true,
      maxLength: 500
    }
  }
}

Options

Common Options

Option Type Default Description
formControlName string Required Name of the form control
label string - Label displayed above the field
placeholder string - Placeholder text
rows number \ string - Number of visible rows
cols number - Number of visible columns
class string - Additional CSS classes
readonly boolean \ function false Whether field is readonly
disabled boolean \ function false Whether field is disabled

Validation Options

Same as Input Field

Examples

Basic Textarea

{
  controlType: 'textarea',
  colSize: 'col-span-12',
  options: {
    formControlName: 'notes',
    label: 'Notes',
    placeholder: 'Enter your notes here...',
    rows: 5
  }
}

Textarea with Character Limit

{
  controlType: 'textarea',
  colSize: 'col-span-12',
  options: {
    formControlName: 'bio',
    label: 'Biography',
    placeholder: 'Tell us about yourself (max 500 characters)',
    rows: 6,
    validators: {
      required: true,
      maxLength: 500,
      minLength: 50
    },
    errorMessage: 'Biography must be between 50 and 500 characters'
  }
}

Auto-Resizing Textarea

{
  controlType: 'textarea',
  colSize: 'col-span-12',
  options: {
    formControlName: 'message',
    label: 'Message',
    rows: 3, // Initial rows
    class: 'resize-vertical' // CSS class for auto-resize
  }
}

Readonly Textarea

{
  controlType: 'textarea',
  colSize: 'col-span-12',
  options: {
    formControlName: 'terms',
    label: 'Terms and Conditions',
    rows: 10,
    readonly: true,
    value: 'Default terms text...'
  }
}

Conditional Textarea

{
  controlType: 'textarea',
  colSize: 'col-span-12',
  options: {
    formControlName: 'customInstructions',
    label: 'Custom Instructions',
    placeholder: 'Add any special instructions...',
    rows: 4,
    visibleWhen: {
      field: 'hasSpecialRequirements',
      value: true,
      operator: 'equals'
    },
    validators: {
      required: true
    }
  }
}
