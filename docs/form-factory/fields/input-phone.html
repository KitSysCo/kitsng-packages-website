Input Phone Field

The input-phone field type provides a phone number input with country code selector and validation.

Basic Usage

{
  controlType: 'input-phone',
  colSize: 'col-span-12',
  options: {
    formControlName: 'phoneNumber',
    label: 'Phone Number',
    placeholder: 'Enter phone number',
    apiBaseUrl: 'https://api.example.com',
    validators: {
      required: true
    }
  }
}

Options

Common Options

Option Type Default Description
formControlName string Required Name of the form control
label string - Label displayed above the field
placeholder string - Placeholder text for phone number input
apiBaseUrl string - Base URL for countries API
readonly boolean \ function false Whether field is readonly
disabled boolean \ function false Whether field is disabled
staticCountriesCode boolean false Use google-libphonenumber for countries if no provider

Examples

Basic Phone Input

{
  controlType: 'input-phone',
  colSize: 'col-span-12',
  options: {
    formControlName: 'phone',
    label: 'Phone Number',
    placeholder: 'Enter phone number',
    apiBaseUrl: 'https://api.example.com',
    validators: {
      required: true
    }
  }
}

Phone with Initial Value

{
  controlType: 'input-phone',
  colSize: 'col-span-12',
  options: {
    formControlName: 'contactPhone',
    label: 'Contact Phone',
    placeholder: 'Enter contact phone',
    apiBaseUrl: 'https://api.example.com',
    value: '+1234567890', // Full international format
    validators: {
      required: true
    }
  }
}

Readonly Phone Display

{
  controlType: 'input-phone',
  colSize: 'col-span-12',
  options: {
    formControlName: 'displayPhone',
    label: 'Phone Number',
    readonly: true,
    value: '+1234567890'
  }
}

How It Works

The phone input field:

  1. Country Code Selector - Dropdown to select country code
  2. Phone Number Input - Text input for the phone number
  3. Automatic Validation - Uses google-libphonenumber for validation
  4. Format Conversion - Stores value in international format (e.g., +1234567890)

Validation

The field automatically validates phone numbers using the google-libphonenumber library. If the phone number is invalid, it adds an invalidPhone error to the form control.

Value Format

The form control value is stored in international format:

  • Format: +[countryCode][number]
  • Example: +1234567890 (USA)
