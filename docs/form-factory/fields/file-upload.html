File Upload Field

The file-upload field type provides file upload functionality with drag-and-drop support and file preview.

Basic Usage

{
  controlType: 'file-upload',
  colSize: 'col-span-12',
  options: {
    formControlName: 'documents',
    label: 'Upload Documents',
    maxFileSize: 1000000, // 1MB
    multiple: true,
    accept: '.pdf,.doc,.docx',
    validators: {
      required: true
    }
  }
}

Options

Common Options

Option Type Default Description
formControlName string Required Name of the form control
label string - Label displayed above the field
maxFileSize number 1000000 Maximum file size in bytes
multiple boolean false Allow multiple file selection
accept string 'image/*' Accepted file types
fileLimit number - Maximum number of files
fileInputMode 'advanced' \ 'basic' 'advanced' File input mode
showUploadMessage boolean false Show drag & drop message
chooseLabel string - Label for choose button
chooseIcon string - Icon for choose button
chooseStyleClass string - CSS class for choose button
apiBaseUrl string - Base URL for previewing uploaded files
onRemoveFile function - Callback when file is removed
readonly boolean \ function false Whether field is readonly
disabled boolean \ function false Whether field is disabled

Examples

Basic File Upload

{
  controlType: 'file-upload',
  colSize: 'col-span-12',
  options: {
    formControlName: 'avatar',
    label: 'Profile Picture',
    maxFileSize: 5000000, // 5MB
    accept: 'image/*',
    multiple: false,
    showUploadMessage: true
  }
}

Multiple Files with Custom Button

{
  controlType: 'file-upload',
  colSize: 'col-span-12',
  options: {
    formControlName: 'attachments',
    label: 'Attachments',
    multiple: true,
    fileLimit: 5,
    accept: '.pdf,.doc,.docx,.jpg,.png',
    chooseLabel: 'Select Files',
    chooseIcon: 'pi pi-paperclip',
    chooseStyleClass: 'p-button-outlined'
  }
}

File Upload with Remove Handler

{
  controlType: 'file-upload',
  colSize: 'col-span-12',
  options: {
    formControlName: 'documents',
    label: 'Documents',
    onRemoveFile: (fileUrl: string, formGroup: FormGroup) => {
      console.log('File removed:', fileUrl);
      // Handle file removal from server if needed
    }
  }
}

Value Format

The form control value will be:

  • A single File object (if multiple: false)
  • An array of File objects (if multiple: true)
  • If loading existing data, it can be a string (URL) or array of strings (URLs).

// Example value (New selection):
File { name: "image.png", size: 12345, ... }


// Example value (Existing data):
"https://example.com/uploads/image.png"
