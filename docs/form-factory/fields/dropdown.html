Dropdown Field

The dropdown field type provides a single-select dropdown with filtering and grouping support.

Basic Usage

{
  controlType: 'dropdown',
  colSize: 'col-span-12',
  options: {
    formControlName: 'country',
    label: 'Country',
    placeholder: 'Select a country',
    dropdownOptions: [
      { id: 1, text: 'United States' },
      { id: 2, text: 'Canada' },
      { id: 3, text: 'United Kingdom' }
    ],
    optionLabel: 'text',
    optionValue: 'id',
    validators: {
      required: true
    }
  }
}

Options

Common Options

Option Type Default Description
formControlName string Required Name of the form control
label string - Label displayed above the field
placeholder string - Placeholder text
dropdownOptions any[] [] Array of options
optionLabel string 'label' Property name for option label
optionValue string 'id' Property name for option value
showClear boolean false Show clear button
filter boolean false Enable filtering
filterBy string 'text' Property to filter by
appendTo string 'body' Where to append dropdown panel
group boolean false Enable option grouping
optionGroupLabel string 'text' Property for group label
optionGroupChildren string 'items' Property for group children
readonly boolean \ function false Whether field is readonly
disabled boolean \ function false Whether field is disabled

Examples

Basic Dropdown

{
  controlType: 'dropdown',
  colSize: 'col-span-12',
  options: {
    formControlName: 'status',
    label: 'Status',
    placeholder: 'Select status',
    dropdownOptions: [
      { id: 'active', text: 'Active' },
      { id: 'inactive', text: 'Inactive' },
      { id: 'pending', text: 'Pending' }
    ],
    optionLabel: 'text',
    optionValue: 'id',
    validators: {
      required: true
    }
  }
}

Dropdown with Filtering

{
  controlType: 'dropdown',
  colSize: 'col-span-12',
  options: {
    formControlName: 'product',
    label: 'Product',
    placeholder: 'Search and select product...',
    dropdownOptions: products, // Large array of products
    optionLabel: 'name',
    optionValue: 'id',
    filter: true,
    filterBy: 'name',
    showClear: true,
    validators: {
      required: true
    }
  }
}

Grouped Dropdown

{
  controlType: 'dropdown',
  colSize: 'col-span-12',
  options: {
    formControlName: 'city',
    label: 'City',
    placeholder: 'Select a city',
    dropdownOptions: [
      {
        label: 'USA',
        items: [
          { id: 1, name: 'New York' },
          { id: 2, name: 'Los Angeles' }
        ]
      },
      {
        label: 'Canada',
        items: [
          { id: 3, name: 'Toronto' },
          { id: 4, name: 'Vancouver' }
        ]
      }
    ],
    group: true,
    optionGroupLabel: 'label',
    optionGroupChildren: 'items',
    optionLabel: 'name',
    optionValue: 'id',
    filter: true
  }
}

Dropdown with Change Handler

{
  controlType: 'dropdown',
  colSize: 'col-span-12',
  options: {
    formControlName: 'category',
    label: 'Category',
    placeholder: 'Select category',
    dropdownOptions: categories,
    optionLabel: 'name',
    optionValue: 'id',
    ngModelChange: (value, formGroup, selectedOption, field) =&gt; {
      console.log('Category selected:', selectedOption);
      // Update dependent fields, load related data, etc.
      if (selectedOption) {
        loadSubcategories(selectedOption.id);
      }
    }
  }
}

Conditional Dropdown

{
  controlType: 'dropdown',
  colSize: 'col-span-12',
  options: {
    formControlName: 'shippingMethod',
    label: 'Shipping Method',
    placeholder: 'Select shipping method',
    dropdownOptions: [
      { id: 'standard', text: 'Standard Shipping' },
      { id: 'express', text: 'Express Shipping' },
      { id: 'overnight', text: 'Overnight Shipping' }
    ],
    visibleWhen: {
      field: 'hasShipping',
      value: true,
      operator: 'equals'
    },
    validators: {
      required: true
    }
  }
}
