Calendar Picker Field

The calendar-picker field type provides date, time, and date-time selection with extensive customization options.

Basic Usage

{
  controlType: 'calendar-picker',
  colSize: 'col-span-12',
  options: {
    formControlName: 'birthDate',
    label: 'Birth Date',
    placeholder: 'Select date',
    calendarType: 'date',
    dateFormat: 'mm/dd/yy',
    showIcon: true,
    validators: {
      required: true
    }
  }
}

Options

Common Options

Option Type Default Description
formControlName string Required Name of the form control
label string - Label displayed above the field
placeholder string - Placeholder text
calendarType 'date' \ 'time' \ 'dateTime' \ 'month' \ 'year' 'date' Calendar type
dateFormat string - Date format (e.g., 'mm/dd/yy', 'dd/mm/yyyy')
hourFormat '12' \ '24' - Hour format for time selection
showIcon boolean true Show calendar icon
minDate Date - Minimum selectable date
maxDate Date - Maximum selectable date
defaultDate Date - Default selected date
firstDayOfWeek number - First day of week (0=Sunday, 1=Monday, etc.)
disabledDates Date[] - Array of disabled dates
disabledDays number[] - Array of disabled weekday numbers
yearRange string - Year range (e.g., '1900:2100')
selectionMode 'single' \ 'multiple' \ 'range' 'single' Selection mode
dataType 'string' \ 'date' 'string' Return type
readonlyInput boolean false Make input readonly
appendTo string 'body' Where to append calendar panel
readonly boolean \ function false Whether field is readonly
disabled boolean \ function false Whether field is disabled

Examples

Date Selection

{
  controlType: 'calendar-picker',
  colSize: 'col-span-12',
  options: {
    formControlName: 'appointmentDate',
    label: 'Appointment Date',
    placeholder: 'Select date',
    calendarType: 'date',
    dateFormat: 'mm/dd/yy',
    showIcon: true,
    minDate: new Date(), // Can't select past dates
    validators: {
      required: true
    }
  }
}

Date with Year Range

{
  controlType: 'calendar-picker',
  colSize: 'col-span-12',
  options: {
    formControlName: 'birthDate',
    label: 'Birth Date',
    calendarType: 'date',
    yearRange: '1900:2030',
    dateFormat: 'dd/mm/yy',
    maxDate: new Date(), // Can't select future dates
    validators: {
      required: true
    }
  }
}

Disabled Weekends

{
  controlType: 'calendar-picker',
  colSize: 'col-span-12',
  options: {
    formControlName: 'workingDate',
    label: 'Working Date',
    placeholder: 'Select date (weekends excluded)',
    calendarType: 'date',
    disabledDays: [0, 6], // Sunday and Saturday
    dateFormat: 'mm/dd/yy',
    validators: {
      required: true
    }
  }
}

Disabled Specific Dates

{
  controlType: 'calendar-picker',
  colSize: 'col-span-12',
  options: {
    formControlName: 'bookingDate',
    label: 'Booking Date',
    placeholder: 'Select available date',
    calendarType: 'date',
    disabledDates: [
      new Date('2024-12-25'), // Christmas
      new Date('2025-01-01')  // New Year
    ],
    minDate: new Date(),
    validators: {
      required: true
    }
  }
}

First Day of Week (Monday)

{
  controlType: 'calendar-picker',
  colSize: 'col-span-12',
  options: {
    formControlName: 'startDate',
    label: 'Start Date',
    calendarType: 'date',
    firstDayOfWeek: 1, // Monday
    dateFormat: 'dd/mm/yyyy',
    validators: {
      required: true
    }
  }
}

Return Date Object Instead of String

{
  controlType: 'calendar-picker',
  colSize: 'col-span-12',
  options: {
    formControlName: 'eventDate',
    label: 'Event Date',
    calendarType: 'date',
    dataType: 'date', // Returns Date object instead of string
    dateFormat: 'mm/dd/yy',
    validators: {
      required: true
    }
  }
}
