Select Button Field

The select-button field type provides button-style selection for single or multiple choices.

Basic Usage

{
  controlType: 'select-button',
  colSize: 'col-span-12',
  options: {
    formControlName: 'size',
    label: 'Size',
    selectButtonOptions: [
      { id: 'small', text: 'Small' },
      { id: 'medium', text: 'Medium' },
      { id: 'large', text: 'Large' }
    ],
    optionLabel: 'text',
    optionValue: 'id',
    multiple: false,
    validators: {
      required: true
    }
  }
}

Options

Common Options

Option Type Default Description
formControlName string Required Name of the form control
label string - Label displayed above the field
selectButtonOptions any[] [] Array of options
optionLabel string 'text' Property name for option label
optionValue string 'id' Property name for option value
multiple boolean false Allow multiple selection
selectbuttonLabelType 'icon' \ 'html' \ null null Label type
readonly boolean \ function false Whether field is readonly
disabled boolean \ function false Whether field is disabled
