Tree Select Field

The tree-select field type provides a tree/hierarchical selection interface for selecting nested options.

Basic Usage

{
  controlType: 'tree-select',
  colSize: 'col-span-12',
  options: {
    formControlName: 'category',
    label: 'Category',
    placeholder: 'Select category...',
    treeSelectOptions: [
      {
        label: 'Electronics',
        children: [
          { label: 'Laptops', data: { id: 1 } },
          { label: 'Phones', data: { id: 2 } }
        ]
      },
      {
        label: 'Clothing',
        children: [
          { label: 'Men', data: { id: 3 } },
          { label: 'Women', data: { id: 4 } }
        ]
      }
    ],
    validators: {
      required: true
    }
  }
}

Options

Common Options

Option Type Default Description
formControlName string Required Name of the form control
label string - Label displayed above the field
placeholder string - Placeholder text
treeSelectOptions TreeNode[] [] Tree node options
selectionMode 'single' \ 'multiple' \ 'checkbox' 'single' Selection mode
display 'comma' \ 'chip' 'comma' Display style for selected items
filter boolean false Enable filtering
metaKeySelection boolean false Use meta key for selection

Examples

Single Selection Tree

{
  controlType: 'tree-select',
  colSize: 'col-span-12',
  options: {
    formControlName: 'location',
    label: 'Location',
    placeholder: 'Select location...',
    treeSelectOptions: locationTree,
    selectionMode: 'single',
    filter: true
  }
}

Multiple Selection Tree

{
  controlType: 'tree-select',
  colSize: 'col-span-12',
  options: {
    formControlName: 'departments',
    label: 'Departments',
    placeholder: 'Select departments...',
    treeSelectOptions: departmentTree,
    selectionMode: 'multiple',
    display: 'chip',
    filter: true
  }
}

Checkbox Selection Tree

{
  controlType: 'tree-select',
  colSize: 'col-span-12',
  options: {
    formControlName: 'permissions',
    label: 'Permissions',
    placeholder: 'Select permissions...',
    treeSelectOptions: permissionTree,
    selectionMode: 'checkbox',
    display: 'chip'
  }
}

Tree Node Format

Tree nodes follow PrimeNG TreeNode structure:

interface TreeNode {
  label: string;
  data?: any;
  children?: TreeNode[];
  expanded?: boolean;
  selectable?: boolean;
}
