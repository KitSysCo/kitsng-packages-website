Checkbox Field

The checkbox field type provides a single checkbox input for boolean values.

Basic Usage

{
  controlType: 'checkbox',
  colSize: 'col-span-12',
  options: {
    formControlName: 'agreeToTerms',
    label: 'I agree to the terms and conditions',
    binary: true,
    validators: {
      requiredTrue: true
    }
  }
}

Options

Common Options

Option Type Default Description
formControlName string Required Name of the form control
label string - Label displayed next to checkbox
binary boolean true If true, uses true/false values; if false, uses checkbox value
value any true Value when checkbox is checked (if binary is false)
class string - Additional CSS classes
readonly boolean \ function false Whether field is readonly
disabled boolean \ function false Whether field is disabled

Validation Options

validators: {
  requiredTrue?: boolean; // For binary checkboxes
  required?: boolean; // For non-binary checkboxes
}

Examples

Binary Checkbox (True/False)

{
  controlType: 'checkbox',
  colSize: 'col-span-12',
  options: {
    formControlName: 'isActive',
    label: 'Active',
    binary: true,
    value: true
  }
}

Non-Binary Checkbox

{
  controlType: 'checkbox',
  colSize: 'col-span-12',
  options: {
    formControlName: 'notifications',
    label: 'Enable Email Notifications',
    binary: false,
    value: 'email-notifications'
  }
}

Required Checkbox (Terms Acceptance)

{
  controlType: 'checkbox',
  colSize: 'col-span-12',
  options: {
    formControlName: 'acceptTerms',
    label: 'I accept the Terms and Conditions',
    binary: true,
    validators: {
      requiredTrue: true
    },
    errorMessage: 'You must accept the terms to continue'
  }
}

Conditional Checkbox

{
  controlType: 'checkbox',
  colSize: 'col-span-12',
  options: {
    formControlName: 'newsletter',
    label: 'Subscribe to newsletter',
    binary: true,
    visibleWhen: {
      field: 'hasAccount',
      value: true,
      operator: 'equals'
    }
  }
}

Disabled Checkbox

{
  controlType: 'checkbox',
  colSize: 'col-span-12',
  options: {
    formControlName: 'premiumFeature',
    label: 'Premium Feature Access',
    binary: true,
    disabled: (form: FormGroup) => {
      // Disable if user is not premium
      return form.get('accountType')?.value !== 'premium';
    }
  }
}

Checkbox with Change Handler

{
  controlType: 'checkbox',
  colSize: 'col-span-12',
  options: {
    formControlName: 'enableTwoFactor',
    label: 'Enable Two-Factor Authentication',
    binary: true,
    ngModelChange: (value, formGroup, selectedOption, field) => {
      if (value) {
        // Show additional setup fields
        showTwoFactorSetup();
      } else {
        // Hide setup fields
        hideTwoFactorSetup();
      }
    }
  }
}
