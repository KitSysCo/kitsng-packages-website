Password Field

The password field type provides a secure input for entering passwords with optional strength meter and visibility toggle.

Basic Usage

{
  controlType: 'password',
  colSize: 'col-span-12',
  options: {
    formControlName: 'password',
    label: 'Password',
    placeholder: 'Enter password',
    validators: {
      required: true,
      minLength: 8
    }
  }
}

Options

Common Options

Option Type Default Description
formControlName string Required Name of the form control
label string - Label displayed above the field
placeholder string - Placeholder text
feedback boolean true Show password strength meter
toggleMask boolean true Show show/hide password icon
showClear boolean false Show clear icon
promptLabel string - Label for prompt
weakLabel string - Label for weak password
mediumLabel string - Label for medium password
strongLabel string - Label for strong password
size 'small' \ 'large' - Size of the input
fluid boolean false Spans 100% width of container
variant 'filled' \ 'outlined' - Input variant
autocomplete string 'off' HTML autocomplete attribute
readonly boolean \ function false Whether field is readonly
disabled boolean \ function false Whether field is disabled

Examples

Basic Password

{
  controlType: 'password',
  colSize: 'col-span-12',
  options: {
    formControlName: 'password',
    label: 'Password',
    feedback: false // Hide strength meter
  }
}

Password with Strength Meter

{
  controlType: 'password',
  colSize: 'col-span-12',
  options: {
    formControlName: 'newPassword',
    label: 'New Password',
    feedback: true,
    promptLabel: 'Enter a password',
    weakLabel: 'Too simple',
    mediumLabel: 'Average complexity',
    strongLabel: 'Complex password',
    validators: {
      required: true,
      minLength: 8
    }
  }
}

Outlined Variant

{
  controlType: 'password',
  colSize: 'col-span-12',
  options: {
    formControlName: 'confirmPassword',
    label: 'Confirm Password',
    variant: 'outlined',
    toggleMask: true
  }
}
