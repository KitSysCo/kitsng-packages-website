Multi-Select Field

The multi-select field type provides a dropdown for selecting multiple values with chip or comma display.

Basic Usage

{
  controlType: 'multi-select',
  colSize: 'col-span-12',
  options: {
    formControlName: 'tags',
    label: 'Tags',
    placeholder: 'Select tags',
    multiSelectOptions: [
      { id: 1, text: 'Technology' },
      { id: 2, text: 'Design' },
      { id: 3, text: 'Business' },
      { id: 4, text: 'Marketing' }
    ],
    optionLabel: 'text',
    optionValue: 'id',
    display: 'chip',
    validators: {
      required: true
    }
  }
}

Options

Common Options

Option Type Default Description
formControlName string Required Name of the form control
label string - Label displayed above the field
placeholder string - Placeholder text
multiSelectOptions any[] [] Array of options
optionLabel string 'text' Property name for option label
optionValue string 'id' Property name for option value
display 'chip' \ 'comma' 'chip' Display style for selected items
showToggleAll boolean false Show select all/none toggle
filter boolean false Enable filtering
filterBy string 'text' Property to filter by
showClear boolean false Show clear button
group boolean false Enable option grouping
optionGroupLabel string 'text' Property for group label
optionGroupChildren string 'items' Property for group children
appendTo string 'body' Where to append dropdown panel
readonly boolean \ function false Whether field is readonly
disabled boolean \ function false Whether field is disabled

Dynamic Multi-Select Options

{
  controlType: 'multi-select',
  colSize: 'col-span-12',
  options: {
    formControlName: 'relatedProducts',
    label: 'Related Products',
    placeholder: 'Select related products',
    dependsOn: ['category'],
    dynamicOptions: (form: FormGroup) =&gt; {
      const categoryId = form.get('category')?.value;
      if (!categoryId) return [];
      
      // Return products for selected category
      return getProductsByCategory(categoryId);
    },
    optionLabel: 'name',
    optionValue: 'id',
    display: 'chip',
    filter: true
  }
}

Multi-Select with Min/Max Selection

{
  controlType: 'multi-select',
  colSize: 'col-span-12',
  options: {
    formControlName: 'features',
    label: 'Features (Select 2-5)',
    placeholder: 'Select 2-5 features',
    multiSelectOptions: features,
    optionLabel: 'name',
    optionValue: 'id',
    display: 'chip',
    validators: {
      required: true,
      custom: (control) =&gt; {
        const value = control.value;

          return { required: true };
        }
        if (value.length  5) {
          return { maxSelection: 'Please select at most 5 features' };
        }
        return null;
      }
    }
  }
}
