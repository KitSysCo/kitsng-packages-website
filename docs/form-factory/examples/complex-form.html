Features Demonstrated

  • ✅ Field Dependencies - Subcategory depends on Category selection
  • ✅ Conditional Visibility - Weight/File Size based on Product Type
  • ✅ Conditional Readonly - Stock Quantity readonly for services
  • ✅ Dynamic Options - Subcategory options update based on category
  • ✅ Custom Validators - Price and images validation
  • ✅ Lifecycle Hooks - onInit and ngModelChange handlers
  • ✅ Multiple Field Types - Input, Dropdown, Radio, Slider, Editor, etc.
  • ✅ Dynamic Calculations - Discounted price auto-calculation
  • ✅ File Upload - Product images with validation
  • ✅ Complex Validation - Multiple validation rules per field

Form Flow

  1. User selects Category → Subcategory options update
  2. User selects Product Type → Weight or File Size appears
  3. User enters Price and Discount → Discounted Price auto-calculates
  4. User selects Product Type as "Service" → Stock Quantity becomes readonly
  5. All fields validate independently and on form submission
