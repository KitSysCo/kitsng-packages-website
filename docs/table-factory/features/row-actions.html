Row Actions

Configure action buttons for each row in the table.

Overview

Row actions allow users to interact with individual records. You can configure them using the actionButtons property in KitsngTableConfig.

Basic Usage

tableConfig: KitsngTableConfig = {
  columns: [...],
  data: [...],
  actionButtons: [
    {
      icon: 'pi pi-pencil',
      label: 'Edit',
      onClick: (item) => this.edit(item)
    },
    {
      icon: 'pi pi-trash',
      label: 'Delete',
      colorClass: 'p-button-danger',
      onClick: (item) => this.delete(item)
    }
  ]
};

Configuration Options

The KitsngTabelAction interface supports the following options:

Option Type Description
label string Button label text
icon string Icon class (e.g., 'pi pi-user')
iconPos 'left' \ 'right' \ 'top' \ 'bottom' Icon position
onClick (event: any) => void Click handler function
buttonClass string CSS class for the button
colorClass string Color class (e.g., 'p-button-danger')
tooltip string Tooltip text
type 'Edit' \ 'Delete' \ 'View' Predefined action type
template TemplateRef Custom template for the button

Examples

Icon Only Buttons

actionButtons: [
  {
    icon: 'pi pi-pencil',
    tooltip: 'Edit User',
    onClick: (item) => this.edit(item)
  }
]

Text Only Buttons

actionButtons: [
  {
    label: 'View Details',
    buttonClass: 'p-button-outlined',
    onClick: (item) => this.view(item)
  }
]

Custom Styling

actionButtons: [
  {
    label: 'Approve',
    icon: 'pi pi-check',
    colorClass: 'p-button-success',
    onClick: (item) => this.approve(item)
  },
  {
    label: 'Reject',
    icon: 'pi pi-times',
    colorClass: 'p-button-danger p-button-outlined',
    onClick: (item) => this.reject(item)
  }
]

Using Predefined Types

actionButtons: [
  {
    type: 'Edit',
    onClick: (item) => this.edit(item)
  },
  {
    type: 'Delete',
    onClick: (item) => this.delete(item)
  }
]

Custom Action Template

For complete control over the action column, you can use actionTemplate in KitsngTableConfig:

tableConfig: KitsngTableConfig = {
  // ...
  actionTemplate: this.customActionTemplate
};

Bulk Actions

You can also configure actions for multiple selected rows using bulkActions:

tableConfig: KitsngTableConfig = {
  // ...
  showSelectionCheckbox: true,
  showBulkActions: true,
  bulkActions: [
    {
      label: 'Delete Selected',
      icon: 'pi pi-trash',
      colorClass: 'p-button-danger',
      onClick: (selectedItems) => this.deleteMultiple(selectedItems)
    }
  ]
};
