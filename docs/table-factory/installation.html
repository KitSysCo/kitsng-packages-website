Installation & Setup

This guide will help you install and configure kitsng-table-factory in your Angular project.

Prerequisites

  • Angular 20.3.0+
  • PrimeNG 20.3.0+ (for table components)
  • RxJS
  • TypeScript

Installation

Step 1: Install the Package

npm install kitsng-table-factory

Or if using a local package:

npm install ./path/to/kitsng-table-factory

Step 2: Install Peer Dependencies

The library requires the following peer dependencies:

npm install primeng primeicons @angular/common @angular/core @angular/forms rxjs
Required versions:
  • primeng: ^20.3.0
  • primeicons: ^7.0.0

Step 3: Import the Module

In your Angular module (e.g., app.module.ts):

import { NgModule } from '@angular/core';
import { BrowserModule } from '@angular/platform-browser';
import { CommonModule } from '@angular/common';
import { HttpClientModule } from '@angular/common/http';


// PrimeNG Modules (required)
import { TableModule } from 'primeng/table';
import { PaginatorModule } from 'primeng/paginator';
import { ButtonModule } from 'primeng/button';
import { TooltipModule } from 'primeng/tooltip';
import { CheckboxModule } from 'primeng/checkbox';



// kitsng-table-factory
import { KitsngTableFactoryModule } from 'kitsng-table-factory';



@NgModule({
  imports: [
    BrowserModule,
    CommonModule,
    HttpClientModule, // Required if using fetch-table-factory
    // PrimeNG modules
    TableModule,
    PaginatorModule,
    ButtonModule,
    TooltipModule,
    CheckboxModule,
    // kitsng-table-factory
    KitsngTableFactoryModule
  ],
  // ...
})
export class AppModule { }

Step 4: Import Styles

Option 1: Import in angular.json 
{
  "styles": [
    "node_modules/primeng/resources/themes/your-theme/theme.css",
    "node_modules/primeng/resources/primeng.min.css",
    "node_modules/primeicons/primeicons.css",
    "node_modules/kitsng-table-factory/styles.css",
    "styles.css"
  ]
}
Option 2: Import in styles.css 
/ PrimeNG Styles /
@import '~primeng/resources/themes/your-theme/theme.css';
@import '~primeng/resources/primeng.min.css';
@import '~primeicons/primeicons.css';


/ Kitsng Table Factory Styles /
@import '~kitsng-table-factory/styles.css';

Option 3: Import in component 
@Component({
  selector: 'app-users',
  styleUrls: [
    '../../node_modules/kitsng-table-factory/styles.css'
  ]
})

Step 5: Basic Usage

In your component:

import { Component } from '@angular/core';
import { KitsngTableConfig } from 'kitsng-table-factory';


@Component({
  selector: 'app-users',
  template: 
    
  
})
export class UsersComponent {
  tableConfig: KitsngTableConfig = {
    columns: [
      { field: 'id', header: 'ID' },
      { field: 'name', header: 'Name' },
      { field: 'email', header: 'Email', sortable: true }
    ],
    data: [
      { id: 1, name: 'John Doe', email: 'john@example.com' },
      { id: 2, name: 'Jane Smith', email: 'jane@example.com' }
    ]
  };
}

Standalone Components

If you're using standalone components:

import { Component } from '@angular/core';
import { CommonModule } from '@angular/common';
import { KitsngTableFactoryModule } from 'kitsng-table-factory';
import { KitsngTableConfig } from 'kitsng-table-factory';


@Component({
  selector: 'app-users',
  standalone: true,
  imports: [
    CommonModule,
    KitsngTableFactoryModule
  ],
  template: 
    
  
})
export class UsersComponent {
  tableConfig: KitsngTableConfig = {
    columns: [
      { field: 'id', header: 'ID' },
      { field: 'name', header: 'Name' }
    ],
    data: [
      { id: 1, name: 'John Doe' }
    ]
  };
}

Required PrimeNG Modules

The library requires these PrimeNG modules:

Module Purpose
TableModule Main table component
PaginatorModule Pagination controls
ButtonModule Action buttons
TooltipModule Tooltips on buttons
CheckboxModule Row selection checkboxes
TreeTableModule Tree table component (if using tree-table)
RippleModule Button ripple effects

Optional Dependencies

  • HttpClientModule: Required only if using kitsng-fetch-table-factory with generic API
  • FormsModule: Required for column visibility toggle checkboxes

CSS Framework

The library uses Tailwind CSS utility classes with full dark mode support. Make sure you have Tailwind CSS installed and configured in your Angular project. The components work seamlessly with Tailwind utility classes.

Verification

To verify the installation:

import { KitsngTableConfig } from 'kitsng-table-factory';


// Should not show any errors
const config: KitsngTableConfig = {
  columns: [],
  data: []
};

Troubleshooting

Module not found

Ensure you've imported KitsngTableFactoryModule in your module:

imports: [KitsngTableFactoryModule]

Styles not applied

Make sure you've imported the styles.css file in your angular.json or styles.css.

PrimeNG components not rendering

Verify that all required PrimeNG modules are imported in your module.
